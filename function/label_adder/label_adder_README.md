# label_adder

`label_adder`는 Extract_Feature에서 뽑은 csv에 **라벨을 추가**하는 함수입니다. 

이 함수는 머신 러닝 모델 학습에 필요한 라벨을 csv 데이터에 추가할 때 유용합니다.

## 실행
### 실행 방법
```cmd
python main.py
```
